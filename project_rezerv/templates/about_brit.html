БРИТ

БРИТ применяется совместно с прибором ПУ/ПУПТ и содержит всю необходимую индикацию и органы управления (кнопки) для 4-х направлений пожаротушения. Не заменяет собой БДУ.

Детальнее про прибор

Технические характеристики:

Напряжение питания - 12В, 0.1А.
Ток потребления в дежурном режиме - не более 10 мА + 3 мА на каждый включенный индикатор.
Индикация:
       светодиодный индикатор <ПУСК> (красный) 1шт. х 4.
       светодиодный индикатор <Стоп АСПТ> (желтый) 1шт. х 4.
       светодиодный индикатор <Отказ АСПТ> (желтый) 1шт. х 4.
       светодиодный индикатор <Блокировка> (желтый) 1шт. х 4.
       светодиодный индикатор <Норма> (зеленый) 1шт. х 4.
       светодиодный индикатор <РУЧН/АВТ> (желтый) 1шт. х 4.
       светодиодный индикатор <ПОДТВЕРДИТЕ ПУСК> (красный) 1шт.
       светодиодный индикатор <Питание> (зеленый) 1шт.
Управление:
       кнопка управления <ПУСК> 1шт х 4.
       кнопка управления <РУЧН/АВТ> 1шт. х 4.
       кнопка управления <ПОДТВЕРДИТЕ ПУСК> 1шт.
Коммуникация - внутриситемный канал связи RS485: A1/B1 - основной, A2/B2 - резервный.
Габаритные размеры в мм (Ш х В х Г) - 235 х 275 х 80.
